Projeto escrito em linguagem de programação Delphi XE 11, que consiste em uma aplicação de diagnóstico técnico para apoio em análises efetuadas pelo time de atendimento ao cliente.
A aplicação avalia as seguintes informações na base do cliente : hardware, ambiente, banco de dados, parametros do sistema, rede, etc.
Todo resultado da análise é exibido em tela para o operador no componente ListBox, chamado LSTPRINCIPAL.
Todas as rotinhas de checagem, são iniciadas a partir do botão btnAnalise.
